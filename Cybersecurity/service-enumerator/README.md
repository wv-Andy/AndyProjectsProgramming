# Service Enumerator (HTTP/HTTPS):
Minimal service enumeration tool that fetches HTTP/HTTPS headers using a HEAD request.

## Usage:
python enumerator.py scanme.nmap.org 80
python enumerator.py scanme.nmap.org 443

## What it extracts:
Server
X-Powered-By
Via
Date
